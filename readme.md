## Panadería Doble J – Página Web

    Proyecto realizado para la asignatura de Desarrollo Web en Entorno Cliente.
    Sitio web completo para una panadería realista, con diseño moderno y funcional.

## Funcionalidades principales
    Página de inicio (index.html)

    Logo y nombre de la panadería en gran formato.

    Eslogan principal centrado.

    Tarjetas de acceso a:

    Panes

    Postres

    Cafés

    Sección “Nuestra historia”.

    Mapa y datos de contacto.

## Página de Panes (panes.html)

    Grid de productos con imágenes uniformes.

    Listado de precios y productos adicionales.

    Tarjetas visuales con título y precio.

## Página de Postres (postres.html)

    Cuadrícula 2×2 de postres.

    Imágenes recortadas y unificadas.

    Diseño limpio y centrado.

## Página de Cafés (cafes.html)

    Sección introductoria.

    Bloque central con foto grande de granos y 4 cajas de colores a los lados.

    Diseño similar a una infografía sin bordes.

## Página de Pedidos (pedido.html)

    Selector de cantidad con botones + y –.

    Botón ADD funcional (front-end).

    Formulario para:

    Nombre

    Hora de recogida

    Botón final BUY NOW.

## Header y navegación

    Header fijo con:

    Logo pequeño

    Nombre de la panadería

    Botones a Panes / Postres / Cafés

    Rutas corregidas para que todo funcione desde subcarpetas.

## Diseño general

    Fondo degradado en tonos café.

    Sombras suaves y bordes redondeados.

    Tipografía Poppins.

    Uso de variables CSS para colores y sombras.
